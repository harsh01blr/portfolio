# Project_Portfolio
A comprehensive collection of simple data modeling and scripting projects that demonstrate enterprise solution design skills using open-source tools.


## Skills Demonstrated
- Data Modeling: ERDs, relational schema design (MySQL, SQLite)
- Scripting: Python (Flask, FastAPI, Pandas)
- Analytics: Streamlit dashboards, data visualization
- AI/ML: scikit-learn, OpenCV for medical imaging
- Cloud & DevOps: AWS boto3, scaling simulation
- Security: Log monitoring, intrusion detection script

Core Objectives:
1.	Deploying RAG Pipelines for Production at Scale
2.	Decentralized AI at Scale with NVIDIA Flare
3.	AI for Monitoring, Prediction, and Automation for Large Scale Systems
4.	AI Model Training at Scale
5.	Data Curation, Model Customization, and Large-Scale Training
